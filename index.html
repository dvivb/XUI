XUI Contains variety of interface elements and control

  • Basic controls: basic interface unit, used to build web applications such as buttons, text labels, input boxes, etc.
  • Components: by more than one of the basic control of complex interface unit.Such as form, menus, forms, etc.
  • Javascript components: these components to normal use will use Javascript.Some call to enable Javascript components need to manually.
  • View: the view is the basic control and component combinations used to show the general content of website, such as comment on the view and the article preview list view.
